Collage Web App

Using Collage Web App you can create a collage of the selfies you take. The collage will contain 3 selfies, all of which will be taken using this web app. The selfies you take will be displayed below.


Click the start button before speaking the command. A popup will appear asking for your permission; click Allow. Then speak your command. Whatever you speak will appear in the given area. Make sure to speak only and only "take my selfie" and no other alternative.

After you've spoken the command, when another popup appears, click Allow. It will allow this web app to take a selfie.
